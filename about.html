About This Website

This website, Voices for Palestine, was created to raise awareness and support for the Palestinian cause. It allows users from around the world to show their solidarity by voting and viewing global support statistics.

As a beginner in web development, I built this project to improve my skills while contributing to a meaningful cause. I hope this platform inspires others to stand for justice and humanity.
